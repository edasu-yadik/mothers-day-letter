# Mothers Day Letter 💗

An interactive Mother's Day surprise website built with HTML, CSS and JavaScript.

## Features
- Animated floating hearts
- Envelope opening animation
- Typewriter letter effect
- Rotating memory gallery
- Responsive mobile design
- Background music
- Final cinematic message

## Live Demo
https://edasu-yadik.github.io/mothers-day-letter/

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Project Purpose
This project was created as a handmade digital Mother's Day gift.
